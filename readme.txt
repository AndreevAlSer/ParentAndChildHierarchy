1.Устанавливаем программную платформу Node.js на локальный компьютер.
2.Для выполнения тестового задания я использовал Javascript + Node.js+SQL
3.В папке с программой присутствуют видео файлы, которые показывают, как работает приложение.
4.Устанавливаем SQL Server Management Studio 2017 для работы с базой данных.
5.В файле db_handler.js определен объект, в котором определены параметры для подключения к базе данных. Среди которых: user, password, server, database.
6.Их необходимо будет изменить в зависимости от того, как настроено подключение к локальному серверу с помощью Sql Server Management Studio
7.Это касается пользователя, пароля подключения, имени сервера, имени базы данных.
8.В папке с приложением находится файл database.sql, в котором  находится код для работы с базой данных.
9.В Sql Server Management Studio необходимо создать новый запрос, и вставить туда код из файла database.sql. Весь код необходимо выполнить, следуя подробным комментариям, которые находятся в файле. 
10.Файлы приложения находятся в папках api и app. Папки можно скопировать в любую созданную свою папку. 
11.Файлы приложения (5 шт) открыть в любой IDE. Я использовал Visual Studio Code.
12.В корневую созданную свою папку, где находятся папки api, app необходимо установить необходимые модули для работы программы. 
13.Это можно сделать, открыв консоль ctrl + J в случае использования Visual Studio Code
14.Необходимо ввести следующие команды для установки необходимых модулей:
npm i express
npm i path
npm i body-parser
npm i http
npm i mssql
15.Приложение и сервер работаю на разных портах, поэтому для запуска приложения необходимо выполнить следующие команды в папках api и app.
c:\ElifTech\api> node main
c:\ElifTech\app> node main
16.Запускаем браузер, приложение будет работать по адресу htttp://localhost:8081.
17.Видео инструкция работы приложения прилагается.
